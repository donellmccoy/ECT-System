DBSign Installation Page
Why Am I Here?
You have reached this page because you want to use digital signing with a CAC or personal certificate. However, DBsign must be installed first.

Please note: Installing DBsign requires administrator privileges. If you do not have administrator privileges on your computer or are unsure, please contact your site administrator and have him return to this page: link to perform the installation.

What is DBsign? DBsign is security software that is used in LOD to authenticate users and to enable them to digitally sign the LOD. DBsign includes a Web browser control that supports the access of a user's credentials stored on, for example, a floppy or a Common Access Card.

How do I install DBsign? Please read the following instructions before starting installation.
  1. Select this link (DBsignWebSigner.msi) to save or run the DBsign installation program. If you save the installation program to your hard disk, you'll need to remember its location and run it after saving.


  2. Run the installation program (as Administrator). Accepting the default settings will be sufficient for most users.


  3. Restart your browser after installing DBsign. (Make sure to close all browser windows)


  4. DBsign is now configured properly and should be ready for use.


  5. You may verify the integrity of the downloaded DBsignWebSigner.exe by verifying the MD5 hash value. The Microsoft File Checksum Integrity Verifier (FCIV) utility can be used for this purpose. This tool may be obtained from http://support.microsoft.com/kb/841290
